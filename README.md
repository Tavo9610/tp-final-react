# React/Frontend Final Project - Proyecto Final React/Frontend 
## Summary / Descripcion
#### This project is the final part of the first repository (Whatsapp UI), it allows the chats to be functional, its navigable and the entire structure is using React-Vite to give it functionality while as usual HTML/CSS structures gives the basic tags and styles
#### Este proyecto es la parte final del primer repositorio(UI de Whatsapp), permite darle funcionalidad, navegar entre rutas, y la estructura entera usa React-Vite para otorgarle la funcionalidad mientras que las estructuras HTML/CSS permiten otorgarle etiquetas y estilos
